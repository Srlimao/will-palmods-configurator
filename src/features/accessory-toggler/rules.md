# Accessory Toggler Feature Context Map

This folder implements the configuration user interface for the Accessory Toggler mod in Palworld.

## Components Structure
- **`AccessoryConfig.jsx`**: The main component orchestrating state, keybinds, background colors, and border style parameters.
- **`styles/accessory.module.css`**: Scoped styles for the panels, forms, color-pickers, sliders, and buttons.
